#  Pollo Asiático - Menú Interactivo

Aplicación web desarrollada con HTML y JavaScript que permite visualizar un menú de platos asiáticos, consultar información de contacto y navegar entre diferentes secciones de forma dinámica.

---

##  Descripción

Pollo Asiático es una aplicación web diseñada para presentar el menú de un restaurante especializado en comida asiática. El sitio permite a los usuarios conocer los platos disponibles, consultar información del establecimiento y acceder a recursos complementarios como ubicación y contenido multimedia.

El proyecto fue desarrollado como práctica de programación web, aplicando conceptos de HTML, JavaScript, manipulación del DOM y organización de contenido interactivo dentro de una interfaz sencilla y fácil de usar.

---

##  Características

* Menú interactivo de platos.
* Navegación dinámica entre secciones.
* Información de contacto.
* Integración con Google Maps.
* Integración de contenido multimedia mediante YouTube.
* Generación dinámica de contenido utilizando JavaScript.
* Interfaz sencilla y fácil de navegar.
* Ejecución directa desde el navegador.

---

##  Tecnologías Utilizadas

* HTML5

##  Funcionalidades Principales

### Página de Inicio

Presenta información general sobre el restaurante y permite acceder rápidamente a las demás secciones de la aplicación.

### Menú de Platos

Muestra una lista de platos asiáticos generados dinámicamente desde un arreglo de datos definido en JavaScript.

### Navegación Dinámica

Permite cambiar entre las secciones de inicio, menú y contacto sin necesidad de recargar la página.

### Ubicación

Integra un enlace directo a Google Maps para facilitar la ubicación del establecimiento.

### Contenido Multimedia

Incluye un video informativo o promocional incrustado desde YouTube.

### Gestión de Datos con JavaScript

Utiliza arreglos y funciones para construir dinámicamente el contenido mostrado al usuario.
